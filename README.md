# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/awanishmishra642-commits/portfolio-awanish.git
```

Move into project

```bash
cd portfolio-awanish
```

Install dependencies

```bash
pnpm install
```

Run locally

```bash
pnpm dev
```

Open

```
http://localhost:3000
```
